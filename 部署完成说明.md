## 🚀 下一步操作

### 方式一：使用一键脚本（推荐）

```bash
./deploy.sh
```

### 方式二：手动操作

#### 1. 上传到 GitHub

```bash
# 初始化 Git（如果还没有）
git init
git add .
git commit -m "Initial commit: AI Text Paraphrasing Tool"
git branch -M main

# 在 GitHub 创建仓库后，替换下面的地址
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

#### 2. 部署到 Vercel

1. 访问 https://vercel.com/new
2. 选择你的 GitHub 仓库
3. 配置环境变量：
   - `VITE_DEEPSEEK_API_KEY`: 你的 DeepSeek API Key
   - `VITE_DEEPSEEK_API_URL`: https://api.deepseek.com/v1/chat/completions (可选)
   - `VITE_DEEPSEEK_MODEL`: deepseek-chat (可选)
4. 点击 Deploy

   - ⚠️ 永远不要将 `.env` 文件提交到 Git

2. **环境变量命名**
   - 所有环境变量必须以 `VITE_` 开头
